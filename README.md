## 変更履歴

### v1.7.2
- 動作ログからチャンネル名のない行の `[]` 表記を除去
- 動作ログから時刻表記を除去
- 種別トグル（動画・Short・ライブ）を編集モード中のみ変更可能に変更
- Ctrl押下中のMinWidthを330pxに制限（従来は完全解除）
- Ctrl非押下中のリサイズで最小サイズを下回った場合即座に強制復元
- ABOUTページに「Created by Coco」を追加
- チャンネルリストの×ボタンを編集モード中のみ表示
- 基本設定のサブナビを横並び（アプリ・監視・ログ・API・ABOUT）に変更
- 基本設定のチェック間隔を 3/5/10/15/20/30/60分 に変更、クォータ超過項目を選択不可に
- ログ保持期間を 0日（当日のみ）・1日・3日・5日・7日・無制限 に変更
- 終了時に LastCheckedVideoId を保存、起動時チェックで新着を検出・通知
- 「確認リスト」を「チャンネルリスト」に名称変更
- サイドバー展開幅を140pxに変更

### v1.7.0
- 「確認リスト」を「チャンネルリスト」に名称変更
- サイドバー展開幅を 120px → 140px に変更
- 動作ログページからエラーログセクションを削除
- 基本設定を縦サブナビゲーション → 横並びサブナビゲーション（アプリ・監視・ログ・API・ABOUT）に変更
- 基本設定ページにタイトル行を追加
- チェック間隔を 3分・5分・10分・15分・20分・30分・60分 に変更
  - 起動時にチャンネル数でクォータ計算を行い、超過する間隔を選択不可に設定
- ログ保持期間を 0日（当日のみ）・1日・3日・5日・7日・無制限 に変更
  - 0日設定時は当日分のログのみ保持
- 終了時に各チャンネルの LastCheckedVideoId を保存
- 起動時チェックで前回終了後の新着動画を検出・通知

### v1.6.0
- 基本設定に通知音オプションを追加（デフォルト ON）
  - `Resources\notification.wav` がある場合はそのファイルを25%音量で再生
  - ない場合は Windows の `notify.wav` を25%音量で再生（NAudio使用）
  - デスクトップ通知OFF + 通知音ON → 音のみ再生
- トースト通知クリック時の動作をアイコンクリックと同一に変更
  - 種別フィルターに従い最新動画を開く・NEWバッジを自動消去
- サイドバーの「監視停止」ボタンを削除し、監視中/停止中バッジをトグルボタン化
  - クリックで監視開始/停止・動作ログページへ自動遷移
- サイドバー左下にピンボタンを追加（常に前面表示トグル）
  - 通常時はテーマカラー・ピン留め時は赤色表示
- 基本設定から「常に前面に表示」項目を削除（ピンボタンで代替）
- APIキー入力欄をマスク表示に変更
  - 保存済み時は●マスク表示（コピー不可）・変更ボタンで編集可能に
- チャンネル追加ウィンドウのサイズを 520×190px 固定に変更
- チャンネル行左端にNEWバッジを移動（アイコン左に横書き表示）
- 基本設定にコンパクトモードを追加（ダークモードの下）
  - ON時: アイコン（通常の1/2サイズ）＋チャンネル名のみ表示、行高36px
  - OFF時: 通常表示（行高60px）
- 編集ボタン押下後の完了ボタン押下で元のスタイルに正しく復帰するよう修正
- チャンネル行左端のNEWバッジを削除（アイコンを左詰めに変更）
- 基本設定にコンパクトモードを追加（ダークモードの下）
  - ON時: アイコン（通常の1/2サイズ）＋チャンネル名のみ表示・行高36px
  - OFF時: 通常表示（行高60px）
- 基本設定の各オプションのデフォルト値をOFFに変更（デスクトップ通知・タスクトレイ格納・通知音）
- APIキー未設定時の初回起動でセットアップウィンドウを表示（メインウィンドウはAPIキー保存後に表示）
- チャンネルリストの「今すぐチェック」「編集」ボタンをアイコンのみに変更（ToolTipでラベル表示）
- 初回起動時（config.json未存在）のウィンドウ横幅を405pxに設定
- チャンネルアイコンのディスクキャッシュを実装
  - 保存先: `%APPDATA%\YTNotifier\icons\` （URLのSHA1ハッシュをファイル名に使用）
  - 起動時はディスクキャッシュから読み込み（ネットワーク通信なし）
  - 未キャッシュ時はバックグラウンドでダウンロードして保存・自動反映
- コンパクトモード時に行全体クリックで最新動画を開く（通常モードのアイコンクリックと同一動作）
- コンパクトモード時も左端のAccentBrushバーで新着通知を表示
- 通知音の動作
  - `Resources\notification.wav` がある場合はそのファイルを25%音量で再生
  - ない場合は Windows の `notify.wav` を25%音量で再生
  - 音量制御に NAudio ライブラリを使用
- トースト通知と通知音を独立制御
  - トースト ON + 通知音 ON: トースト表示 + notify.wav 再生
  - トースト ON + 通知音 OFF: トースト表示のみ（無音）
  - トースト OFF + 通知音 ON: notify.wav 再生のみ
  - トースト OFF + 通知音 OFF: 何もしない
- トースト通知クリック時の動作をアイコンクリックと同一に変更
  - 種別フィルターに従い最新動画を開く
  - NEWバッジを自動消去

### v1.5.0

- 基本設定にログメンテナンス機能を追加

  - 自動削除トグル（起動時に古いログを削除）
  - 保持期間設定（1/3/5/7/14/30/60/90日・無制限）
  - 手動削除ボタン・フォルダを開くボタン
- 動作ログページから「フォルダを開く」ボタンを削除（ログメンテナンスに移動）
- チャンネルリストに編集モードを追加
  - 編集ボタン押下中のみドラッグアンドドロップで並び替え可能
  - 編集モード中はアイコンクリック・種別トグルを無効化

  - トースト通知の重複クリック問題を修正（ハンドラを1回のみ登録）
